# Курсовая работа по дисциплине "Базы данных"

## Тема
Клуб любителей собаководства

## Описание
База данных предназначена для учета пород собак, владельцев,
родственных связей, участия в выставках и истории заболеваний.

## Используемая СУБД
PostgreSQL

## Структура базы данных
В базе данных реализованы следующие таблицы:
- breeds
- dogs
- parents
- exhibitions
- diseases
- disease_history

Все таблицы содержат ограничения:
- PRIMARY KEY
- FOREIGN KEY
- UNIQUE
- CHECK
- DEFAULT
